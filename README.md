# sample-backend

A Clojure backend server to use for some graphql testing. Based on the code provided by the Lacinia Tutorial

## Setup

To be able to run the server you need to download Leiningen, one of the clojure tools to handle projects.

## Usage

Run `lein run` and the server will start, making available the point `http://localhost:8888"
